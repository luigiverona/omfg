# Omfg 0.1.2

This patch release completes the public bootstrap trust contract:

- the immutable `install` release asset embeds the literal SHA-256 of its runtime archive and never downloads a checksum to establish trust;
- the deterministic release pipeline builds the archive first, renders and validates the installer from those exact bytes, and publishes four independently attested assets;
- release-note version and content checks now run during the release-contract preflight, before a tag can reach publication; and
- Pages verifies and deploys the exact immutable release installer through an explicit main-branch manual dispatch.

The v0.1.1 Mullvad package correction, AUR output containment, actionable package diagnostics, and Codex installer environment isolation are retained unchanged.
