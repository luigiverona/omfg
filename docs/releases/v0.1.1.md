# Omfg 0.1.1

This patch release corrected the first production workstation installation failures:

- Mullvad VPN moved to the official signed Arch `mullvad-vpn` package, with `mullvad-vpn-daemon` resolved transitively;
- AUR requirements became independent deterministic transactions that exclude unrelated debug and split-package outputs;
- package-manager failures gained concise diagnostics and private full-output logs; and
- the official Codex installer moved into a replacement environment that contains its profile mutation, preserves unrelated Codex installations, reuses valid managed binaries, and checks both profiles independently.

The immutable v0.1.1 installer verifies its archive using adjacent checksum metadata. Version 0.1.2 supersedes that trust mechanism with a checksum embedded directly in the immutable installer asset.
