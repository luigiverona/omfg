# Omfg 0.2.1

This corrective release limits both the custom domain and the default GitHub Pages site to the
single `/install` bootstrap endpoint. Immutable runtime archives, checksum manifests, and
attestations remain exclusively in GitHub Releases.

The generated installer downloads `omfg-0.2.1.tar.gz` directly from the exact immutable `v0.2.1`
GitHub Release tag. Its embedded archive SHA-256 remains the authoritative integrity check; no
network checksum or mutable release lookup is used.

The command interface and workstation setup behavior are unchanged from v0.2.0.
