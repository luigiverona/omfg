# Omfg 0.1.3

This patch release improves terminal output without changing package selection or security boundaries:

- bootstrap downloads are quiet and installation operations report concise results;
- setup uses clear unnumbered stages with compact spacing and state-aware messages;
- sudo displays its native password prompt without a synthetic label;
- interruption reports accurately identify completed and remaining stages; and
- normal setup output no longer includes a version, operating-system, or shell banner.

Verbose mode retains environment, command, subprocess, log, and temporary-workspace diagnostics.
