# Omfg 0.1.0

The first `omfg` release provides a complete, flag-selectable Arch Linux workstation setup:

- strict application and dependency catalogs with deterministic planning;
- supported full-system pacman updates, temporary unprivileged AUR builds, and user-scoped Flatpak/Flathub installation;
- Git identity defaults, GitHub CLI browser authentication, and SSH protocol configuration;
- guarded Ed25519 GitHub key creation with existing keys preserved by default;
- one official Codex executable with isolated `codex-01` and `codex-02` profiles;
- idempotent PATH configuration for fish, Bash, and Zsh;
- mutation-free dry runs and independent final-state verification; and
- a home-directory bootstrap with an immutable layout and published SHA-256 checksums.

Version 0.1.0 supports Arch Linux on x86-64 and must be run as a normal user with sudo access. Automated tests exercise planning, safety boundaries, configuration rendering, release construction, and non-mutating flows. They do not comprehensively exercise real package installation, GitHub/Codex authentication, or destructive workstation mutation.
